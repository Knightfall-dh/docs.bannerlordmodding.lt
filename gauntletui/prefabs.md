# Prefabs

    /GUI/Prefabs/.xml

Defines various GUI elements, such as windows, screens, buttons, etc.

In the game "Movies" are constructed from Prefabs and ViewModel datasource.

Prefabs - groups of [Widgets](/gauntletui/widgets)

## Structure

``` xml
<Prefab>
    <Constants>
        <!-- Insert Summary Here -->
    </Constants>
    <Variables>
        <!-- Insert Summary Here -->
    </Variables>
    <VisualDefinitions>
        <!-- Insert Summary Here -->
    </VisualDefinitions>
    <Window>
        <!-- Insert Summary Here -->
    </Window>
</Prefab>
```
